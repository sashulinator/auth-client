### Subtree

This repo has a subtree repository

```
git subtree [push, pull] --prefix=src/common git@github.com:savchenko91/auth-common.git <branch>
```

## Сущности

### Comp

1. Введеное понятие являющееся сокращением от Components
2. Представляет собой набор свойств
   a. Свойства полей находятся в корне
   b. Свойства компонента находятся в props
